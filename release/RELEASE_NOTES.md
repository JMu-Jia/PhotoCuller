# Photo Culler v1.0.0

Initial Windows portable release.

## Highlights

- Local JPG/JPEG photo culling with optional same-name NEF pairing.
- Pick copies JPG/JPEG + NEF to `PICK/`.
- Delete moves JPG/JPEG + NEF to `_TRASH/`.
- Dynamic thumbnail strip for large folders.
- Original-image based fit preview, zoom, and pan rendering.
- Explicit end marker after the last image.

## Download

Download `PhotoCuller-Windows-Portable-v1.0.0.zip`, extract it, and run:

```bat
Start_Photo_Culler.cmd
```
